# DevOps Capstone Project

## Description

Customer Accounts Microservice Capstone

This project is a RESTful Customer Accounts microservice developed as part of a DevOps Capstone. It demonstrates the end-to-end implementation of modern software development practices, including Agile planning, test-driven development (TDD), continuous integration and delivery (CI/CD), and containerized deployment.

The service is built using Flask and provides core account management operations such as create, read, update, delete, and list. The project incorporates automated testing with high coverage, secure coding practices, and CI pipelines using GitHub Actions. It is containerized with Docker and deployed to a Kubernetes/OpenShift environment, with a Tekton pipeline enabling automated continuous delivery.

This capstone showcases the complete lifecycle of building, testing, securing, and deploying a cloud-native microservice.
